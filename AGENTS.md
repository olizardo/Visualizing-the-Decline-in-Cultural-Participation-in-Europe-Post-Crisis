# Agent Instructions

This repository serves as a reproducibility and replication package. 
When interacting with this codebase, please adhere to the following rules:

1. **Read-Only Data:** Do not modify the original datasets inside the data folder.
2. **Reproducibility:** Ensure any updates to the analysis are captured in the Quarto (.qmd) files so they can be fully reproduced.
3. **Environment:** Use the `renv` package to manage dependencies.
4. **Output:** Direct all generated figures to the `Plots/` directory and tables to `Tabs/`.
